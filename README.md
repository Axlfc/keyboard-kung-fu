## Keyboard-Kung-Fu: Compilation of Keyboard Shortcuts

### Essential Shortcuts (Windows)

- **Select all**: `Ctrl + A` (`Ctrl + E`)
- **Copy text**: `Ctrl + C`
- **Cut text**: `Ctrl + X`
- **Paste text**: `Ctrl + V`
- **Undo action**: `Ctrl + Z`
- **Redo undo action**: `Ctrl + Y`
- **Create new folder or new document or window**: `Ctrl + Shift + N`
- **Rename file/folder**: `F2`
- **Delete selected item**: `Ctrl + D`
- **Delete selected item skipping Recycle Bin**: `Shift + Del`
- **Close active window**: `Alt + F4`
- **Refresh current window**: `Ctrl + R`
- **View open apps**: `Ctrl + Alt + Tab`
- **Switch app**: `Alt + Tab`
- **Switch app backwards**: `Alt + Shift + Tab`
- **Cycle through open windows**: `Alt + Esc`
- **Cycle through open windows backwards**: `Alt + Shift + Esc`
- **Take screenshot**: `Windows + Shift + S`
- **Open File Explorer**: `Windows + E`
- **Open Task Manager**: `Ctrl + Shift + Esc`
- **Open Search**: `Windows + S`
- **Open Settings**: `Windows + I`
- **Open Clipboard bin**: `Windows + V`
- **Open emoji panel**: `Windows + .`
- **Multiple purpose keyboard combination**: `Ctrl + Alt + Del`
- **Open Run command**: `Windows + R`
- **Change web browser tab**: `Ctrl + Tab`
- **Change web browser tab backwards**: `Ctrl + Shift + Tab`
- **Close web browser tab**: `Ctrl + W`
- **Refresh browser tab**: `F5`
- **Zoom in (magnifier)**: `Windows + +`
- **Zoom out (magnifier)**: `Windows + -`
- **Exit magnifier**: `Windows + Esc`

### Desktop Shortcuts

- **Open Start menu**: `Windows`
- **Change Start menu size**: `Ctrl + Arrow keys`
- **Open hidden menu**: `Windows + X`
- **Opens the System Properties dialog box**: `Windows + Pause/Break`
- **Open favorite app**: `Windows + Number (0-9)`
- **Display and hide Desktop**: `Windows + D`
- **Temporarily peek at the desktop**: `Windows + ,`
- **Stretch desktop window to the top and bottom of the screen**: `Windows + Shift + Up arrow key`
- **Maximize or minimize active desktop windows vertically while maintaining width**: `Windows + Shift + Down arrow key`
- **Move active window to monitor on the left**: `Windows + Shift + Left arrow key`
- **Move active window to monitor on the right**: `Windows + Shift + Right arrow key`
- **Snap app or window left**: `Windows + Left arrow key`
- **Snap app or window right**: `Windows + Right arrow key`
- **Maximize app windows**: `Windows + Up arrow key`
- **Minimize app windows**: `Windows + Down arrow key`
- **Open search for the device on a domain network**: `Windows + Ctrl + F`
- **Show System Properties dialog box**: `Windows + Ctrl + Pause`
- **Lock computer**: `Windows + L`

### Text Manipulation

- **Move the cursor to the beginning of the previous word**: `Ctrl + Left arrow key`
- **Move the cursor to the beginning of the next word**: `Ctrl + Right arrow key`
- **Move the cursor to the beginning of the previous paragraph**: `Ctrl + Up arrow key`
- **Move the cursor to the beginning of the next paragraph**: `Ctrl + Down arrow key`
- **Select block of text**: `Ctrl + Shift + Arrow key`
- **Select multiple items**: `Shift + Arrow keys`
- **Move cursor one page up**: `Page Up`
- **Move cursor one page down**: `Page Down`
- **Search**: `Ctrl + F`
- **Scroll to the top**: `Ctrl + Home`
- **Scroll to the bottom**: `Ctrl + End`
- **Convert selected text to bold**: `Ctrl + B`
- **Convert selected text to italics**: `Ctrl + I`
- **Convert selected text to underline**: `Ctrl + U`
- **Save**: `Ctrl + S`
- **Print**: `Ctrl + P`
