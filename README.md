# keyboard-kung-fu
Recopilation of keyboard key combinations & shortcuts to achieve different things (Why we need a mouse anyway?)

## Esssential shortcuts (Windows, simple ones can work in Linux)

#### Select all
<code>Ctrl+a</code>

#### Copy text
<code>Ctrl+c</code>

#### Cut text
<code>Ctrl+x</code>

#### Paste text
<code>Ctrl+v</code>

#### Undo action
<code>Ctrl+z</code>

#### Redo undo action
<code>Ctrl+y</code>

#### Create new folder
<code>Ctrl+Shift+n</code>

#### Rename file/folder
<code>F2</code>

#### Delete selected item
<code>Ctrl+d</code>

#### Delete selected item skipping Recycle Bin
<code>Shift+Supr</code>

#### Close active window
<code>Alt+F4</code>

#### Refresh current window
<code>Ctrl+r</code>

#### View open apps
<code>Ctrl+Alt+Tab</code>

#### Switch app
<code>Alt+Tab</code>

#### Switch app backwards
<code>Alt+Shift+Tab</code>

#### Cycle through open windows
<code>Alt+Esc</code>

#### Cycle through open windows backwards
<code>Alt+Shift+Esc</code>

#### Take screenshot
<code>Windows+Shift+s</code>

#### Open File Explorer
<code>Windows+e</code>

#### Open Task Manager
<code>Ctrl+Shift+Esc</code>

#### Open Search
<code>Windows+s</code>

#### Open Settings
<code>Windows+i</code>

#### Open Clipboard bin
<code>Windows+v</code>

#### Open emoji panel
<code>Windows+.</code>

#### Multiple purpose keyboard combination
<code>Ctrl+Alt+Supr</code>

#### Open Run command
<code>Windows+r</code>

#### Change web browser tab
<code>Ctrl+Tab</code>

#### Change web browser tab backwards
<code>Ctrl+Shift+Tab</code>

#### Close web browser tab
<code>Ctrl+w</code>

#### Refresh browser tab
<code>F5</code>

#### Zoom in (magnifier)
<code>Windows+`+`</code>

#### Zoom out (magnifier)
<code>Windows+`-`</code>

#### Exit magnifier
<code>Windows+Esc</code>

## Text manipulation

#### Move the cursor to the beginning of the previous word
<code>Ctrl+Left arrow key</code>

#### Move the cursor to the beginning of the next word.
<code>Ctrl+Right arrow key</code>

#### Move the cursor to the beginning of the previous paragraph
<code>Ctrl+Up arrow key</code>

#### Move the cursor to the beginning of the next paragraph
<code>Ctrl+Down arrow key</code>

#### Select block of text
<code>Ctrl+Shift+Arrow key</code>

#### Select multiple items
<code>Shift+Arrow keys</code>

#### Move cursor one page up
<code>Page Up</code>

#### Move cursor one page down
<code>Page Down</code>

#### Search
<code>Ctrl+f</code>

#### Scroll to the top
<code>Ctrl+Home</code>

#### Scroll to the bottom
<code>Ctrl+End</code>

#### Convert selected text to bold
<code>Ctrl+b</code>

#### Convert selected text to italics
<code>Ctrl+i</code>

#### Convert selected text to underline
<code>Ctrl+u</code>

#### Save
<code>Ctrl+s</code>
