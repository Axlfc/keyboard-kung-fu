# keyboard-kung-fu
Recopilation of keyboard key combinations & shortcuts to achieve different things (Why we need a mouse anyway?)

## Esssential shortcuts (Windows, simple ones can work in Linux)

#### Select all
<code>Ctrl + a</code>

#### Copy text
<code>Ctrl + c</code>

#### Cut text
<code>Ctrl + x</code>

#### Paste text
<code>Ctrl + v</code>

#### Undo action
<code>Ctrl + z</code>

#### Redo undo action
<code>Ctrl + y</code>

#### Create new folder or new document or window
<code>Ctrl + Shift + n</code>

#### Rename file/folder
<code>F2</code>

#### Delete selected item
<code>Ctrl + d</code>

#### Delete selected item skipping Recycle Bin
<code>Shift + Del</code>

#### Close active window
<code>Alt + F4</code>

#### Refresh current window
<code>Ctrl + r</code>

#### View open apps
<code>Ctrl + Alt + Tab</code>

#### Switch app
<code>Alt + Tab</code>

#### Switch app backwards
<code>Alt + Shift + Tab</code>

#### Cycle through open windows
<code>Alt + Esc</code>

#### Cycle through open windows backwards
<code>Alt + Shift + Esc</code>

#### Take screenshot
<code>Windows + Shift + s</code>

#### Open File Explorer
<code>Windows + e</code>

#### Open Task Manager
<code>Ctrl + Shift + Esc</code>

#### Open Search
<code>Windows + s</code>

#### Open Settings
<code>Windows + i</code>

#### Open Clipboard bin
<code>Windows + v</code>

#### Open emoji panel
<code>Windows + .</code>

#### Multiple purpose keyboard combination
<code>Ctrl + Alt + Del</code>

#### Open Run command
<code>Windows + r</code>

#### Change web browser tab
<code>Ctrl + Tab</code>

#### Change web browser tab backwards
<code>Ctrl + Shift + Tab</code>

#### Close web browser tab
<code>Ctrl + w</code>

#### Refresh browser tab
<code>F5</code>

#### Zoom in (magnifier)
<code>Windows + `+`</code>

#### Zoom out (magnifier)
<code>Windows + `-`</code>

#### Exit magnifier
<code>Windows + Esc</code>
